## Initial installation

### Install client:

[https://cloud.google.com/sdk/docs/install?hl=fr#deb](https://cloud.google.com/sdk/docs/install?hl=fr#deb)

### Then:

```shell
# authenticate
gcloud auth application-default login

# set quota project
gcloud auth application-default set-quota-project snappy-byte-430513-i2
```
